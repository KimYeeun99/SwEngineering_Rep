## 프로젝트 소개
> 이 프로젝트는 캠핑장 예약 관리 프로그램을 소프트웨어 공학 수업에서 배운 절차에 맞춰 제작하는 프로젝트이다.  
> 캠핑장 예약 관리 프로그램은 캠핑장 주인은 캠핑장 정보를 업로드 및 요청된 예약을 관리하고, 손님은 캠핑장 정보 조회 및 예약 요청을 할 수 있는 시스템으로 손쉽게 캠핑장 예약 관리를 할 수 있도록 돕는 시스템이다.
